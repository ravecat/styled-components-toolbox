## Label

A simple text label.

### API

| Property    | Description                                  | Type    | Default |
| ----------- | -------------------------------------------- | ------- | ------- |
| _bold_      | set the bold font style                      | boolean | `false` |
| _labelFor_  | ID of the form element that the label is for | string  | `-`     |
| _uppercase_ | convert text to uppercase                    | boolean | `false` |
